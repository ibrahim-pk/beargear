# 🛒 MERN Grocery Shop Project

Welcome to the MERN BearGear Project! This project is built using the MERN (frontend=> React) stack and incorporates various features.

  ## BearGear E-commerce Website Features

- 🚀 **Login/Sign Up**: Users can create accounts and log in to manage their profiles and orders.
- 🔒 **Update Info**: Users can update their personal information for better security and accuracy.
- 📝 **Customer Reviews**: Customers can leave reviews for products they have purchased.
- ✏️ **Update Reviews**: Customers can edit or delete their reviews.
- 📦 **Product Listings**: Browse and view a wide range of bags.
- 🔍 **Filter Products**: Filter products by category, price, brand, and other attributes.
- 🔎 **Search Products**: Search for specific bags by name or keywords.
- 🛒 **Cart Items**: Add bags to the cart for purchase.
- 🛍️ **Order Summary**: Review and confirm orders before checkout.
- 📄 **Order Details**: View order history and details.
- 💵 **Cash on Delivery**: Payment system supporting cash on delivery for all purchases.
- 👨‍💼 **Admin Functionality**: Special features for administrators to manage products, users, and orders.
- 📱 **Mobile Responsiveness**: Ensure the website is mobile-friendly for easy access on smartphones and tablets.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ibrahim-pk/beargear.git
   ```

2. Change into the project directory:

   ```bash
   cd beargear
   ```

3. Install frontend dependencies:

   ```bash
   npm install
   ```

## Usage


1. Start the frontend:

   ```bash
   cd beargear
   npm start
   ```

2. Access the application in your web browser at `http://localhost:3000`.
